type error =
| Invalid_kind_of_block (*
This error appears when the kind of the current block is wrong.
*)
| Invalid_complement_of_length (*
This error appears when we compute a stored block and length do not correspond with the complement of length.
*)
| Invalid_dictionary (*
This error appears when we compute a dynamic block and we catch an error when we try to decode the dictionary.
*)

The checksum of the output produced does not equal with the checksum of the stream.

val eval : 'a Decompress.B.t ->
       'a Decompress.B.t ->
       ('a, 'a) t ->
       [ `Await of ('a, 'a) t
       | `End of ('a, 'a) t
       | `Error of ('a, 'a) t * error
       | `Flush of ('a, 'a) t ]
eval i o t computes the state t with the input i and the output o. This function returns:
  • `Await t: the state t waits a new input, may be you use Decompress.INFLATE.refill.
  • `Flush t: the state t completes the output, may be you use Decompress.INFLATE.flush.
  • `End t: means that the deflate algorithm is done in your input. May be t writes something in your output. You can check with Decompress.INFLATE.used_out.
  • `Error (t, exn): the algorithm catches an error exn.

val to_result : 'a Decompress.B.t ->
       'a Decompress.B.t ->
       ('a Decompress.B.t -> int) ->
       ('a Decompress.B.t -> int -> int) ->
       ('a, 'a) t ->
       (('a, 'a) t, error) Pervasives.result
to_result i o refill flush t is a convenience function to apply the inflate algorithm on the stream refill and call flush when the internal output is full (and need to flush).

If the compute catch an error, we returns Error exn (see Decompress.INFLATE.error). Otherwise, we returns the state useless t.
